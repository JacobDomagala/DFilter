# DFilter
Simple Sobel edge detection, Gaussian blur and grayscale filters written in CUDA and OpenGL as a part of my Bachelor's thesis.
## Required libraries:
- OpenGL (rendering stuff)
- SDL (creating window)
- SOIL (loading image)
- CUDA (computing stuff)
